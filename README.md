# Student Management System (Java)

A menu-driven CLI-based Student Management System built using Core Java.

## Features
- Add Student
- View Students
- Search Student by ID
- Update Student
- Delete Student

## Technologies Used
- Java
- ArrayList
- Scanner

## How to Run
1. Open project in VS Code
2. Run Main.java

## What I Learned
- Core Java fundamentals
- OOP concepts
- CRUD operations using ArrayList
- Menu-driven program design
